About the Game

This Tic Tac Toe game was created as a fun browser-based version of the classic pencil-and-paper game. Designed using HTML, CSS, and JavaScript, it's a project by a passionate computer engineering student eager to blend coding with creativity.

Game Rules

  • The game is played on a 3x3 grid.
  • Player 1 is X, Player 2 is O. Players take turns marking the empty squares.
  • The first player to align three of their marks vertically, horizontally, or diagonally wins.
  • If all squares are filled without a winner, the game is a draw.
